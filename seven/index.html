HTML 5 & CSS (How to create an effective page layout?)





HTML 5 is a revision of the Hypertext Markup Language (HTML), the standard programming language for describing the contents and appearance of Web pages.

HTML5 establishes a series of new elements and attributes that reflect the typical use of modern websites. Some of them are technically similar to div and span tags, but have semantic meaning, such as nav (website navigation block) and footer.
