HTML stands for Hyper Text Markup Language

HTML is the standard markup language for creating Web pages

HTML describes the structure of a Web page

HTML consists of a series of elements

HTML elements tell the browser how to display the content

HTML elements label pieces of content such as "this is a heading", "this is a paragraph", "this is a link", etc.

CSS




CSS stands for Cascading Style Sheets

CSS describes how HTML elements are to be displayed on screen, paper, or in other media CSS saves a lot of work.

It can control the layout of multiple web pages all at once.

External stylesheets are stored in CSS files.

CSS is used to define styles for your web pages, including the design, layout and variations in display for different devices and screen sizes. 



  

/* CSS files add styling rules to your content */

body {
  font-family: "Benton Sans", "Helvetica Neue", helvetica, arial, sans-serif;
  margin: 2em;
}

p {
    padding-left: 16px
}

h2 {
  font-size: 24px;
  padding-left: 16px
}


body {
  margin: 0;
  font-family: Arial, Helvetica, sans-serif;
  
}

.topnav {
  overflow: hidden;
  background-color: #333;
}

.topnav a {
  float: left;
  color: #96e6b3;
  text-align: center;
  padding: 14px 16px;
  text-decoration: none;
  font-size: 17px;
}

.topnav a:hover {
  background-color: red;
  color: black;
}

.topnav a.active {
  background-color: #96e6b3;
  color: white;
}

HTML Zoom In




HTML is not a programming language; it is a markup language that defines the structure of your content. HTML consists of a series of elements that you will use to enclose different pieces of content so that they look or behave in a certain way. Enclosing tags can make a word or image a hyperlink to another site, change words to italics, make fonts larger or smaller, and so on.

It is very important to consider that each tag requires opening and closing, e.g.. 

    
< body >

 Hello World! 

< /body >

CSS Zoom In




Like HTML, CSS (Cascading Style Sheets) is not really a programming language, nor is it a markup language. It is a style sheet language, that is, it allows you to selectively apply styles to elements in HTML documents. For example, to select all the paragraph elements in an HTML page and turn the text inside them red, you have to write this CSS: 

    
p {
  color: red;
}

To Summarize




The creation of a web page requires at least two files, HTML and CSS, from here we can relink our CSS to multiple pages to create a web site.

In more developed sites this same structure will continue to be used, sometimes extended through programming languages (i.e, JavaScript, Python, etc), or through the use of frameworks, which are HTML and CSS interpreters that enable certain automation and content administration tasks.

However, do not forget the maxim of design "less is more", so let's start by mastering this basic interaction between HTML and CSS to build on these foundations.
